### - OpenAI di React

1. Hal-hal yang harus dipersiapkan
-	Pemahaman terkait react
-	Dapat menggunakan git dan github/sejenisnya
-	Menginstall NPM/Yarn
-	Mempunyai code editor
-	Laptop dan internet
-	Tekat yang kuat untuk belajar

2. Pemasangan OpenAI di React
- Membuat react project
- Install open AI package
- Import module yang diperlukan
- Deklarasi object
- Deklarasi state
- Deklarasi handleclick
- Rendering komponen
